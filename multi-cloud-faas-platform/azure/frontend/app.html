Media Search and Management

Use the following tools to search existing media, analyze a temporary query file, update tags, or delete stored files.

Find Files by Tags

Search files by required species tags and minimum counts. The query uses AND logic, so all listed tag conditions must be satisfied.

Example: {"koala": 3, "wombat": 2} returns files with at least 3 koalas and at least 2 wombats.
